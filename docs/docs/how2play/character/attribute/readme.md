体质（Strength）：代表身体素质和灵巧性。
智慧（Intelligence）：代表学习、记忆和理解能力。
感知（Perception）：代表对外部世界和他人的知觉和掌控能力。
灵能（Super Power）：代表超自然能力。
统帅（Leadership）：代表演讲和说服力，以及对舆论和人心的控制能力。
管理（Management）：代表领导和项目统筹的能力。
