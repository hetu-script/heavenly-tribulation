### 非战斗属性

魅力（charisma）：容貌、气质、说服力和领导力的综合。
悟性(wisdom)：代表学习、记忆和理解能力。
运气（luck）：采集到的资源数量，提升境界成功率等。

### 属性对于角色初始值的影响

每 25 点灵力 + 1 点灵气上限（人物初始为 1）。
体魄和身法之和每增加 4，增加 1 点体力（初始为 10）
每 2 点体魄增加 1 点生命上限（初始为 25）
每 25 点神识可以让主角在地图上的视野范围+1（初始为 1）
