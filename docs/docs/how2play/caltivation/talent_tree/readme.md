# 天赋盘

游戏中升级不能直接加属性点，而是升级修为获得天赋点，通过天赋盘获得属性。

天赋盘是各个流派和辅助功法的被动技能的星图。呈五角星状，对应五个流派：法术、剑修、体修、炼魂、显圣

天赋盘上可以获得属性，也可以直接获得指定卡牌

每个流派附近有两个辅助流派：
