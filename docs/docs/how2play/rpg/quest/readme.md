# 任务

游戏中角色会保存事项，记录了一些重要事件的发展过程。

事项可能有不同的选择分支，以及不同的结局。

任务则是一种特殊事项。除了包含事项都有的过程和结局之外，还包含报酬、预算、时限等额外信息。

## 难度

难度主要对应任务的预算、完成时间等要求。

| difficulty  | 难度 |
| ----------- | ---- |
| easy        | 轻松 |
| normal      | 普通 |
| challenging | 挑战 |
| hard        | 困难 |
| tough       | 艰巨 |
| brutal      | 残酷 |

## 层级

除了难度之外，任务还有一个层级属性。

| 层级 | 对应职级 |
| ---- | -------- |
| 0    | 杂役     |
| 1    | 执事     |
| 2    | 总管     |
| 3    | 堂主     |
| 4    | 长老     |

层级对应了完成任务可以获得的功勋。以及可以领到任务所需要的职级。

悬赏任务的层级最高是 2。

## 悬赏任务 bounty quest

悬赏是在城市会堂处刷新，便是非门派成员也可以领取的一种任务。

门派弟子在完成了指定任务后，也可以继续接取悬赏任务。

虽然不是门派成员，但完成悬赏任务同样可以积累一些功勋。这些功勋在角色后期选择加入此门派时会保留。悬赏任务提供的功勋相比门派任务会比较少。

## 门派任务

任务只有门派成员在每个月 5 日之前参加例会后才会领取到。

例会时，未完成的任务会减少角色在门派中的功勋。例会后上个月的任务会被清空。

门派任务相比悬赏任务，包含了更多的类型。

### 杂役的任务

·「无」提升等级、境界

·「无」探索普通秘境

·「无」在某个建筑中打工 15 天

·交易所：将某些材料从某个城市运送到门派的城市，或者从门派城市运送到另一个城市

·交易所：以一定金钱作为预算，获取指定数量的某种材料

·生产建筑：无预算，要求获取指定数量的某种材料

农场：生产指定数量的粮食、药材
林场：生产指定数量的木材、药材
渔场：生产指定数量的清水、生肉
矿山：生产指定数量的石料、矿物
猎场：生产指定数量的皮革、生肉

·将某个物品送给某个城市的某个角色

### 执事的任务

·提升等级、境界

·探索高阶秘境

·护送某个角色从某个城市到另一个城市

·守卫某个建筑

·获取指定数量和境界的装备、功法、丹药

·探索敌对角色、建筑和门派情报

·交易所：获取某几种指定材料：给与一定数量的金钱，要求购买指定数量的材料。获取方式不限，可以生产、购买。可以去较为便宜的地方购买。剩余的金钱可以自行保留。

·聚灵阵：收集灵光：每个月会在每个门派的地块上刷新一定数量的灵光。可以使用大地图技能「灵视」探查并收集。最终交给门派的聚灵阵。

·藏书阁：制作符纸。

·藏书阁：制作符箓。

·斗技厅：应对挑战者。

·哨所：获取其他势力的某个建筑的情报，守卫某个建筑

·拍卖行：采购法宝：给与一定数量的灵石，要求购买指定类型、稀有度的法宝。程序生成此任务时一定会是目前地图上已有的某个拍卖行中的物品。

·酒楼：打造酒饮。

·阵法所：打造指定的阵法卡牌

·符法所：打造指定的符箓卡牌

·工坊：打造指定的物品

·丹炉：打造指定的药剂

·观星台：通过占卜获取指定情报

·显圣殿：获取神通并在某个地方使用

·幻术堂：识别乔装的奸细

·通灵居：获取指定死者的情报

### 总管的任务

·提升等级、境界

·探索高阶秘境

·获取指定数量和境界的装备、功法、丹药

·守卫某个城市，和入侵者战斗

·探索敌对角色、建筑和门派情报

·破坏某个敌对门派城市

·抢夺敌对势力资源、物品、功法

·刺杀敌对角色

### 堂主的任务

·提升等级、境界

·探索高阶秘境

·获取指定数量和境界的装备、功法、丹药

·探索敌对角色、建筑和门派情报

·抢夺敌对势力资源、物品、功法

·刺杀敌对角色

·花费材料升级某个建筑的规模

·修建某个建筑

·招募某个人加入门派

·和某个门派结交/破坏/结盟/宣战

·跟随某个长老，一同进攻某个其他门派的城市

### 长老的任务

·提升等级、境界

·探索普通秘境

·获取指定数量和境界的装备、功法、丹药

·探索敌对角色、建筑和门派情报

·抢夺敌对势力资源、物品、功法

·刺杀敌对角色

·扩充门派成员达到某个数量

·带领队伍，进攻某个其他门派的城市
