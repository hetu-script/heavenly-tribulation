# 打工和生产

某些建筑可以进行打工赚钱，或者生产材料。

进行这些活动时，会少量增加角色基础属性。

| 场所   | 属性 | 小游戏         |
| ------ | ---- | -------------- |
| 农场   | 神识 | 组合消除       |
| 林场   | 灵力 | 组合消除       |
| 渔场   | 念力 | 组合消除       |
| 猎场   | 身法 | 组合消除       |
| 矿场   | 体魄 | 组合消除       |
| 问道碑 | 悟性 | 大模型猜谜游戏 |
| 酒楼   | 运气 | 赌博游戏       |
