# 打工和生产

某些建筑可以进行打工赚钱，或者生产材料。

进行这些活动时，会少量增加角色基础属性。

| 场所   | 属性 | 小游戏                                   |
| ------ | ---- | ---------------------------------------- |
| 藏书阁 | 悟性 | 拼图                                     |
| 拍卖行 | 魅力 |                                          |
| 酒楼   | 运气 |                                          |
| 农场   | 神识 | 找不同                                   |
| 林场   | 灵力 |                                          |
| 渔场   | 念力 |                                          |
| 猎场   | 身法 |                                          |
| 矿场   | 体魄 | 格子场地，点击获取随机碎片，然后组合消除 |
