# 装备

装备可以为人物提供加成

装备使用的词条系统和天赋盘的技能使用的系统是相同的。

一个角色的总属性等于基础属性+装备提供的属性+天赋提供的属性。

最终他们会合并计算。

# 角色等级和装备
