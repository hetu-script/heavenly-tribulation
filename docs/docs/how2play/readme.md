## RPG

类似恐怖黎明的职业组合和升级系统。

每个角色可以从 5 个主流派和 10 个副流派中任选两种进行组合。可以选两个主流派或者两个副流派。

## 修炼

修炼界面可以进行打坐，此时世界仍会进行演化，但主角将没有参与的机会。

本身是一个类似节奏音游的小游戏，通过点击或者滑动消除界面上浮现的各种光点和光柱来收集经验，经验积累到一定程度就可以直接升级。

## 战斗

战斗是类似弈仙牌的卡牌对战，但不同之处在于在游戏中是通过类似 RPG 的方式收集和升级卡牌
