# 剧情

## 教程剧情

开局时，NPC指引如何提升等级、境界。

凝气境，NPC指引如何加入门派。

劫数接近最大值时，NPC提示如何削减劫数（丹药、装备、提升境界）。

当角色第一次濒死时，NPC提示通过休息恢复体力。

当角色第一次进入拥有鞋或者船的交易所时，一个神秘角色提示主角购买相应装备，并使用装备进入某个原本无法进入的大地图地点。然后触发剧情，获得物品奖励。

## 秘境剧情

秘境本身有一个神秘角色对主角进行引导。

## 建筑剧情

当使用建筑达到一定次数后，再次进入建筑场景，会触发一个建筑系列剧情。整局游戏每种建筑只能触发一次。

## 角色剧情

NPC剧情根据角色种族，性别和流派决定，因此一共有 2 × 3 × 5 = 30 个不同的角色故事。

## 人物关系剧情

根据角色人物关系，玩家会面临一些固定的故事。

## 门派剧情

门派剧情根据门派类型和门派流派决定，因此一共有 5 × 7 = 35 个不同的门派故事。

## 任务剧情（概率）

随机出现在据点会堂的悬赏任务列表中，或者镇魔司的悬赏任务列表中，玩家可以选择接取。如果不接取，之后每个月刷新时仍有几率出现。如果接取过，之后不会再次出现。

## 随机遭遇（概率）

当角色进入隐藏的洞天（山地据点）或福地（岛屿据点）时，触发剧情（概率）。

## 中期挑战

### 大瘟疫

疫情威胁着整个世界。

### 倚天屠龙

新的法宝或者功法被研究出来，其威力可以改变整个世界的平衡格局。

# 终局挑战

### 争锋

几大门派争夺天下霸主

### 魔教

使用恐怖主义和洗脑等手段搅乱天下的魔教。
