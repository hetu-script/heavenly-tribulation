## 路径

| 阶段        | 重点                                 | 依赖                                     |
| ----------- | ------------------------------------ | ---------------------------------------- |
| **Phase 1** | 建筑场景功能（装备打造->灌灵->丹药） | 无，可立即开始                           |
| **Phase 2** | 卡牌/装备词条精简和补充              | 依赖 Phase 1 产出的装备/丹药进入战斗系统 |
| **Phase 4** | 经营策略（生产线/商路/贸易）         | 依赖 Phase 1 的建筑                      |
| **Phase 3** | 门派关系机制                         |                                          |
| **Phase 5** | 剧情内容                             |                                          |

## 关于卡牌、装备和丹药的不同打造体验

三条线的核心动词完全不同：

|          | 卡牌                          | 装备                               | 丹药                      |
| -------- | ----------------------------- | ---------------------------------- | ------------------------- |
| **核心** | 微调                          | 拆装                               | 炼制                      |
| **获取** | 藏书阁 / 问道碑（指定主词条） | 工坊（选类型+附加萃取材料）/灌灵池 | 炼丹炉（选稀有度+主词条） |
| **改造** | 6种材料精细操作               | 无（不满意就拆了重造）             | 无                        |
| **回收** | 分解->碎片->合成打造材料      | 萃取->词条材料->下次制造用         | 无（消耗品用完即止）      |
