# 天道奇劫

A game draws a lot of inspiration from Battlebrothers & Slay the Spire.

这是一个借鉴了很多《太阁立志传》和《弈仙牌》的玩法的游戏。

关于游戏玩法和设定的更多介绍，请参考**docs**目录下的文档。
